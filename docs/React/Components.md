Every web application today has a lot of features. And those are going to be something fundamental for any application. For example Navigation bar, Routes, Login functionality, etc. So, here let's document some of the must know components as a developer. Because, this can save you a lot of time while developing it.

> None of the components below is mandatory, it is just a collection of common components that you would be needing while building a good web application. 

- Login functionality
    - Firebase
    - Oauth2
    - Custom Login functionality (email/mobile & password | OTP based)

- Notifications

    Ability to see the latest notifications.

- Lists

    Ability to list thousands of items either through `Pagination` or `virtualization`

- Filters

- Calendar (Time / Date selection)
